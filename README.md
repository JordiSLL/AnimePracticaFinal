# AnimeRandom
Practica Final de Arquitectura i tecnologia de sistemes web i multimèdia

AnimeRandom es una aplicación que te permite encontrar  animes random en base al genero, puntuación, tipo o estado. Ademas la aplicación incluye un chat general y un "Adivina el Opening/Ending".

## Caracteristicas

### AnimeRandom
### ChatGlobal
### Opening/Ending Guess

**La aplicación hace peticiones a dos Apis diferentes:**
* AnimeRandom: [Jikan.moe](https://jikan.moe/)
* AnimeOpening: [animethemes.moe](https://animethemes.moe/)

## Prerequisites

* Docker Desktop instalado en tu sistema.

## Installation

1. Clona o descarga el repositorio de AnimeRandom de GitHub.
2. Ejecuta el siguiente comando en la ubicacion del Docker-compose.yml: `docker compose up`.
3. En tu navegador accede a `http://localhost:8080`
